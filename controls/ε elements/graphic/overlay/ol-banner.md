## Properties

### Location

The horizontal location of the objects that go within the Overlay placeholder that lays on top of the Graphic element. This container is for all overlays (Top, Center, Bottom).

 - Left (Top)
 - Center (Top)
 - Right (Top)
 - Full width (Top)
 - Center
 - Left (Bottom)
 - Center (Bottom)
 - Right (Bottom)
 - Full width (Bottom)
